Bachelor of Technology (B-Tech)

B-tech

B-Tech (Bachelor of Technology) is a professional undergraduate engineering course. The duration of this course is 4 years and it contains 8 semesters.

Semester 1

  • Engineering Mathematics I
  • Engineering Physics
  • Basic Electrical Engineering
  • Engineering Graphics

Semester 2

  • Engineering Mathematics II
  • Programming in C
  • Basic Electronics
  • Environmental Studies

Semester 3

  • Data Structures
  • Digital Electronics
  • Discrete Mathematics
  • Object Oriented Programming

Semester 4

  • Database Management System
  • Operating System
  • Computer Organization
  • Software Engineering

Semester 5

  • Computer Networks
  • Web Technology
  • Artificial Intelligence

Semester 6

  • Machine Learning
  • Cloud Computing
  • Cyber Security

Semester 7

  • Big Data
  • Internet of Things
  • Project Work

Semester 8

  • Major Project
  • Industrial Training
